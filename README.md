# Toxic Friend a Website to spot manipulation using AI 
Link: https://toxicfriend.netlify.app/ <br>
I made this project using HTML | CSS | JS in frontend, and python Flask and SQLITE in the backend for saving user's state, name and settings.
## Features:
1. It can detect various types of manipulation tactics in the given msg and explain it <br>
2. AI friend with whom we can talk by setting its toxicity from 0 to 100 and it will talk according to its toxicity
3. Premium UI with different themes and personalized experience
4. Clean Structure of the website

## Folder Structure:
Frontend:  Contains all the frontend files and assets <br>
Backend:  Contains all the backend files includes python and sqlite database<br>
Admin Page:  Main Admin Page from which i can see all users and their names<br>
